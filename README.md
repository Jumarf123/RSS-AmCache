# RSS-AmCache

**RSS-AmCache** — GUI‑утилита для парсинга Amcache и проверки найденных файлов по встроенным YARA правилам.

**Запуск:** откройте `rss-amcache.exe` — приложение начнёт парсинг/скан и покажет таблицу с фильтрами и экспортом.

**Фильтры:** Сортируйте по дате, удалению, используйте поиск, можете поставить фильтр, где будут только файлы, на которые есть детект.

**Настройки:** Поддержка Ru + Eng языка.
